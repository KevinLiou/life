一、 選擇油品

常常聽到某些人進行生酮飲食，常常吃到最後好的膽固醇下降、壞的膽固醇上升、三酸甘油脂上升…等等情況。其實是因為太刻意吃油，而且又攝取過量不好的油脂。對於生酮飲食的人來說，選擇吃什麼油是很重要的一環，應該多做點功課。

看完圖表可以知道

  • 我們不應該刻意攝取動物油，照一般飲食的攝取量即可。因為我們會積極吃肉，所以基本上已經攝取足夠的動物性油脂。攝取過量會 造成膽固醇過高。
  • 料理應該選用耐高溫的油品可以，首選富含Omega-9的橄欖油。
  • 建議積極攝取富含Omega-3的油品，每日可以飲用一小匙。因為不耐高溫，所以只適合冷食料理。
  • 上圖是個大概結論，大部分油品都可能 同時含有飽和脂肪酸、Omega-3、Omega-6。只是因為Omega-3攝取量較不足，所以建議積極攝取。以 義美生機100%亞麻仁油250ml 舉例，其實還是有飽和脂肪，只是比例低。

生酮飲食篇(二) 飲食準則

9月 3 2018 生活飲食 7 分钟 read (About 1016 words)

飲食重點

1. 準則一：限制醣分攝取

生酮飲食重點，用最簡潔的話來說，就是『少吃碳水化合物搭配多運動』，體內多餘的脂肪就會被消耗。每一餐的醣含量，應限制在20克以下(每日控制在60克以內)。如果是低醣食品，攝取的醣則不算在內。

註
低糖食品 = 每一百公克中，含醣低於10公克的食品。

2. 準則二：確實的攝取蛋白質

雖然在生酮飲食中，能量的主要來源為脂肪，但攝取蛋白質同樣重要。因為葡萄糖新生(體內缺少葡萄糖，而透過肌肉生產葡萄糖。)會消耗肌肉的氨基酸。因此在實行生酮飲食法時，必須大量攝取蛋白質來維持體內蛋白質水平。

蛋白質為組成人體的材料，積極攝取蛋白質，才能避免肌膚產生明顯的皺紋或鬆弛現象。身體缺乏蛋白質，也容易形成各種副作用。

註

  • 醣類 = 碳水化合物 - 礦物質
  • 每日蛋白質攝取量為：每一公斤體重，需要1.2 ~ 1.6 g
    例如：體重六十公斤的人，每天攝取蛋白質量約為72g ~ 96g

3. 準則三：大量攝取膳食纖維、礦物質

一天應攝取的膳食纖維總量為二十克以上(水溶性+非水溶性)。葉菜類的食物纖維，約佔總重量的3%~5%。因此要達到攝取量，約等於每天要食用400~670克的生鮮葉菜才夠。只要分成三餐，量就沒有那麼多了。

而礦物質的量，每日應攝取：

  • 鉀 3.5克 以上
  • 鈣 650毫克 以上
  • 鎂 350毫克 以上

註

  1. 可以發現，每餐的「肉類、魚類」 約等於 「蔬菜、菇類」。
    例如攝取了200克牛排，那麼蔬菜及菇類也應該攝取200克
  2. 不喜歡吃蔬菜可打成果昔，若沒時間攝取蛋白質，可以在果席內加入蛋白質營養素。

4. 準則四：每天攝取一小匙以上Omega-3脂肪酸

有些油脂無法在體內合成，只能透過飲食攝取，稱為必要脂肪酸；必要脂肪酸包括Omega-3及Omega-6脂肪酸。其中因為Omega-3不易由食物中獲得，因此建議積極攝取。關於油類介紹，可 參考這裡 。

  1. 理想比例
    Omega-3 : Omega-6 = 1 : 4 世界衛生組織的建議則是 1 : 6
  2. 肉類食品中，牧草牛的脂肪酸比例最為理想。
  3. Omega-3脂肪酸僅存在於深海魚與堅果種子類（尤其亞麻仁籽含量最多）。亞麻仁油、任胡麻油、魚油…等，都是富含Omega-3的油品，每天食用一小匙就能達到攝取量。
  4. Omega-3不耐高溫，不適合加熱調理。建議當成沙拉醬料或是混在冷食中食用也不錯。

生酮飲食篇(一) 生酮飲食的優點與常見問題

9月 2 2018 生活飲食 6 分钟 read (About 897 words)

一、引言

  近日有朋友想參加X麗的減肥計畫，經過聊天後發現，減肥計劃中除了昂貴產品(例如：一些蛋白素、卵磷脂、乳酸菌、纖維素、消化酵素…等等)，剩下的飲食計畫也是以低醣為主。這樣的飲食方式上跟生酮飲食有許多相似之處，外加賣你一些昂貴的酵素。參加減肥計劃需要不小的開銷，而且酵素也不可能吃一輩子，計劃之後往往容易復胖。因此最好的方式，還是自己研究飲食方法，並找到自己吃的快樂又健康的方式才是。
 
  自己也即將邁入三十歲，身體新陳代謝速度下降，原本偏瘦的身材也逐漸往中等身材發展。為了維持身體健康與良好的體態，就藉此機會來研究研究生酮飲食。

二、生酮飲食的優點

  1. 不易疲勞、提高注意力、飯後不會想睡、有精神、不再焦躁
  2. 減少體脂肪、改善貧血、便秘、糖尿病、脂肪肝、高血壓、逆流性食道炎、憂鬱、偏頭痛、預防動脈硬化、預防老人癡呆、抗癌…等等 — 來源

三、常見問題

1. 生酮飲食是否危害健康

  1. 愛斯基摩人以肉、魚類為主食，也是生活得好好的。早期的人類以狩獵為食，主要攝取肉類，依然有足夠的能量進行打獵，而後才發展到農耕時代。
  2. 如今生酮飲食的攝取，其實非常多元化。各式蛋類、豆類(豆腐、豆皮、納豆、毛豆)、肉類、魚類、蔬菜……等等，基本上對健康無害。如果拿來跟一般飲食，高鹽、高糖、高澱粉比起來，相較之下還是健康許多。
  3. 糖尿病、肝臟、腎臟病患者不宜嘗試。如果有特殊疾病，實施前也應該先諮詢專業醫師。
  4. 生酮飲食最大的問題，是如何攝取到好的脂肪。飽和脂肪?不飽和脂肪?Omega6與Omega3該如何選擇？可以 參考這裡

2. 如何判斷身體生酮狀態

可購買試紙透過尿液進行測量。

3. 生酮飲食期限

  生酮飲食的效果，因人而異。一開始嘗試，可以兩週為基準，身體無不適則進行下去，最久盡量不超過一個月。其後可以轉為半生酮飲食，攝取少量醣類主食。
